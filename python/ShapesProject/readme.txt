
This project called "Shapes" is a simple demonstration of Python custom classes and how to make them with different attributes.

Files ---

testShapes.py
    This file is a simple test program that will use all 3 "shapes" within the shape pkg.
    It has constructors for all 3 shapes: circle, triangle and rectangle.
    It uses all methods that are owned by these classes.

shape.pkg
    This package "shape" contains the 3 shapes(classes) that are described above.
    circle.py
        Contains the circle class
    triangle.py
        Contains the triangle class
    rectangle.py
        Contains the rectangle class
